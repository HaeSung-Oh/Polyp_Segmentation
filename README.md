## Code for [Effective Colorectal Polyp Segmentation via Shallow Decoder with Global-Local Aggregation and Selective Skip Fusion]

---

## Repository Structure
- `train.py` : Training and validation code
- `eval.py`  : Evaluation metrics (Dice, IoU, PA, Precision, Recall, F1)
- `test.py`  : Testing script using a trained model
- `model.py` : Network architecture
- `dataset.py` : Dataset loader
---

## Requirements
- Python >= 3.8
- PyTorch
- MONAI
- NumPy
